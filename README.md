# 🍔 Swiggy Food Delivery Dashboard

## 📊 Project Overview
This project is an Excel-based dashboard that analyzes Swiggy food delivery data. It provides insights into sales, orders, customer ratings, and performance trends.

## 🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Graphs  
- Data Cleaning  

## 📈 Key Insights
- Daily and weekly order trends  
- Revenue analysis  
- Customer ratings overview  
- Top-performing food categories
